## Notes

### OpenClaw Security Audit Fix

**Problem**: `plugins.installs_unpinned_npm_specs` — three plugins (discord, line, voice-call) used `latest` instead of exact version, supply chain stability insufficient.

**Fix applied:**
```bash
openclaw plugins install @openclaw/discord@2026.5.6 --force
openclaw plugins install @openclaw/line@2026.5.6 --force
openclaw plugins install @openclaw/voice-call@2026.5.6 --force
openclaw gateway restart
```

**Result**: ✅ Three plugins pinned to exact version 2026.5.6, warning eliminated.

**Explanation**: OpenClaw security audit checks if `plugins.installs` field uses exact semver versions. Floating versions like `latest` or `major` fail the audit. After pinning, audit recognizes supply chain as stable.
